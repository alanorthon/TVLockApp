## 🇷🇺 Русский

**TVLockApp** — это специализированное приложение для Android TV (Leanback), предназначенное для родительского контроля и ограничения экранного времени. Оно позволяет устанавливать ежедневные лимиты использования для любого установленного приложения на вашем телевизоре.

### ✨ Основные функции
*   **Ограничение времени:** Установка лимита в часах и минутах для каждого приложения отдельно.
*   **Автоматическая блокировка:** При превышении лимита приложение блокируется, и пользователь перенаправляется на домашний экран.
*   **Защита PIN-кодом:** Настройки и возможность изменения лимитов защищены PIN-кодом.
*   **Адаптация под ТВ:** Интерфейс полностью оптимизирован для управления пультом (D-pad).
*   **Умный сброс:** Лимиты автоматически обновляются каждые сутки (в полночь).
*   **Фоновый мониторинг:** Надежное отслеживание активности через системный сервис.

### 🛠 Технологии
*   **Kotlin + Coroutines:** Для плавной работы и асинхронных операций.
*   **Room Database:** Надежное хранение настроек и лимитов.
*   **Android Leanback SDK:** Современный интерфейс для Android TV.
*   **UsageStatsManager:** Точное отслеживание экранного времени.
*   **Foreground Service:** Гарантирует, что мониторинг не будет остановлен системой.

### 🚀 Установка и настройка
1. Скачайте и установите APK (ссылка ниже).
2. При первом запуске установите мастер-PIN.
3. Предоставьте разрешение на "Доступ к статистике использования" (Usage Stats) в настройках Android.
4. Выберите нужные приложения в списке и установите для них лимиты.

## 🇺🇸 English

**TVLockApp** is a specialized Android TV (Leanback) application designed for parental control and screen time management. It allows you to set daily usage limits for any installed app on your TV.

### ✨ Key Features
*   **Time Limits:** Set specific limits in hours and minutes for each individual app.
*   **Auto-Blocking:** Once the limit is reached, the app is blocked, and the user is redirected to the Home screen.
*   **PIN Protection:** Settings and limit management are secured with a PIN code.
*   **TV Optimized:** UI is fully optimized for remote control (D-pad) navigation.
*   **Smart Reset:** Limits are automatically refreshed every day at midnight.
*   **Background Monitoring:** Reliable activity tracking via a system foreground service.

### 🛠 Tech Stack
*   **Kotlin + Coroutines:** For smooth performance and async operations.
*   **Room Database:** Persistent storage for settings and limits.
*   **Android Leanback SDK:** Modern interface designed for Android TV.
*   **UsageStatsManager:** Precise screen time tracking.
*   **Foreground Service:** Ensures monitoring stays active in the background.

### 🚀 Setup Guide
1. Download and install the APK (link below).
2. Set up your master PIN on the first launch.
3. Grant "Usage Access" permission in Android settings when prompted.
4. Select apps from the list and set your desired daily limits.
